# SecondCortex Human-AI Interaction Policy

## Rules
1. Be transparent about memory scope (project vs all projects).
2. Surface uncertainty and confidence in every assistant answer.
3. Provide at least one recovery action when answer quality is low.
4. Prefer user control before executing impactful actions.
5. Explain why key actions/recommendations are made.

## Required Response Fields
- confidence
- limitations
- nextBestActions
- sources
